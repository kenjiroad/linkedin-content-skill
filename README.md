# LinkedIn Content Skill — Flowmatter

Expert LinkedIn organic content strategy skill for Claude Code.

Built for B2B founders, GTM leaders, and agency owners who want to grow on LinkedIn with a proven system.

## What's Inside

| Sub-Skill | Purpose |
|-----------|---------|
| `hooks` | Write first lines that stop the scroll (210 chars) |
| `storytelling` | AIDA, PAS, BAB, Mistake>Lesson, Contrarian frameworks |
| `cta` | Drive saves, comments, follows — comment gating |
| `formats` | Carousel, video, text-only — reach multipliers |
| `scheduling` | Golden Hour routine, best posting times |
| `engagement` | Comment strategy, DM sequences, platform limits |
| `repurposing` | Content flywheel — 1 post → 6-8 formats |

## Resources

- `resources/voice/flowmatter-writing-guide.md` — Writing voice, tone, formatting rules, pre-publish checklist
- `resources/templates/post-structure-templates.md` — 5 production-ready post templates (A-E)
- `resources/references/linkedin-algorithm.md` — Algorithm mechanics, engagement weights, format performance

## Key Numbers

- Hook = first **210 characters** (before "see more" on mobile)
- Carousels get **2.5-3.5x reach** vs text-only
- Post **3-4x/week** for optimal growth
- Best times: **Tue-Thu, 7:30-8:30 AM**
- First **60-90 minutes** = 80%+ of total reach
- Saves weighted **5x**, meaningful comments **4x**

Once installed, Claude Code auto-invokes the skill from natural language:

- "Turn this post into a carousel" → repurposing + formats
