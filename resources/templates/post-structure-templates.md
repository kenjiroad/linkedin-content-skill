# Post Structure Templates (Production-Ready)

## Template A — Comprehensive Breakdown
**Meilleur pour :** Tech stack reveals, frameworks, how-to guides
**Longueur cible :** 1 100-1 400 caractères

```
[HOOK — 1-2 lignes, <210 caractères]

[CONTEXTE — 1-2 lignes qui légitiment le post]

[SECTION 1]
🔸 [Titre section]

[Ligne 1 explication]
[Ligne 2 explication]
[Ligne 3 si nécessaire]

[SECTION 2]
🔸 [Titre section]

[Ligne 1 explication]
[Ligne 2 explication]

[SECTION 3-5 — même structure]

[SYNTHÈSE — 1-2 lignes qui résument l'ensemble]

[CTA — 1 ligne spécifique]

P.S : [Question ou tease optionnel]
```

---

## Template B — Listicle
**Meilleur pour :** Outils, conseils numérotés, erreurs à éviter
**Longueur cible :** 800-1 100 caractères

```
[HOOK — nombre + promesse, <210 caractères]

[Intro 1 ligne — contexte rapide]

→ [Item 1] — [explication 1 ligne]
→ [Item 2] — [explication 1 ligne]
→ [Item 3] — [explication 1 ligne]
→ [Item 4] — [explication 1 ligne]
→ [Item 5] — [explication 1 ligne]

[Conclusion — 1 ligne forte]

[CTA]
```

---

## Template C — Case Study / Before-After (BAB)
**Meilleur pour :** Résultats clients, transformations, ROI
**Longueur cible :** 900-1 200 caractères

```
[HOOK — le résultat en premier, <210 caractères]

AVANT :
[État initial avec détails spécifiques]
[Chiffre ou situation concrète]

APRÈS :
[État amélioré avec chiffres concrets]
[Résultat mesurable]

LE PONT :
↳ [Étape 1]
↳ [Étape 2]
↳ [Étape 3]

[Leçon ou insight]

[CTA]
```

---

## Template D — Comment-Gated Lead Magnet
**Meilleur pour :** Génération de leads, croissance audience, ressources à partager
**Longueur cible :** 600-900 caractères

```
[HOOK — claim bold + métrique spécifique, <210 caractères]

[Preuve de légitimité — 2-3 lignes]

Voici ce que vous obtenez :
→ [Bénéfice 1]
→ [Bénéfice 2]
→ [Bénéfice 3]
→ [Bénéfice 4]

Commentez "[KEYWORD]" et je vous envoie tout.

[Assurez-vous qu'on est connectés pour que je puisse DM]
```

---

## Template E — Conversational Story (Mistake>Lesson)
**Meilleur pour :** Vulnérabilité, réflexions carrière, histoires personnelles
**Longueur cible :** 900-1 300 caractères

```
[HOOK — confession ou erreur, <210 caractères]

[Ce qui s'est passé — 3-4 lignes narratives avec détails]

[Ce que ça a coûté — chiffre ou impact concret]

[Le moment de bascule — qu'est-ce qui a changé]

[La leçon — formulée pour le lecteur]

[Reframe — pourquoi vous êtes reconnaissant ou ce que vous feriez différemment]

[CTA]

P.S : [Question personnelle au lecteur]
```
