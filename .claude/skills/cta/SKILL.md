# LinkedIn CTAs & Profile Optimization

You are a specialist in crafting LinkedIn call-to-actions that drive saves, comments, and follows, and in optimizing LinkedIn profiles as conversion funnels.

---

## CTA Types by Goal

| Goal | CTA Type | Exemple |
|------|----------|---------|
| **Maximize reach** | Save prompt | "Sauvegardez ce post pour votre prochain [situation]." |
| **Boost engagement** | Comment prompt | "Quel est votre défi #1 avec [sujet] ? Dites-le en commentaire." |
| **Grow followers** | Follow prompt | "Je partage des frameworks [sujet] chaque semaine. Suivez pour la suite." |
| **Generate leads** | DM prompt | "Envoyez-moi '[keyword]' en DM et je vous envoie le template." |
| **Build email list** | Link-in-comments | "J'ai écrit un guide complet sur ça. Lien en commentaire." |

---

## CTA Design Rules

1. **Un seul CTA par post** — plusieurs CTAs diluent l'action
2. **Soyez spécifique** — "Commentez votre plus grande erreur de recrutement" bat "Dites-moi ce que vous pensez"
3. **Abaissez la barrière** — "Déposez un 🔥" est plus facile qu'un paragraphe
4. **Faites correspondre le CTA au contenu** — frameworks = save, opinions = comment, personal brand = follow
5. **Les saves valent 5x** — demandez-vous toujours "est-ce que c'est sauvegardable ?"

## High-Performing CTA Templates

**Save-driving :**
```
"Sauvegardez ce post pour la prochaine fois que vous [situation]. Vous vous remercierez."
```

**Comment-driving :**
```
"Curieux — qu'ajouteriez-vous à cette liste ? Votre #1 en commentaire."
```

**Debate-driving :**
```
"D'accord ou pas ? Je réponds à chaque commentaire."
```

**Follow-driving :**
```
"Je décompose des stratégies [niche] comme ça chaque semaine. Suivez pour la suite."
```

**DM-driving :**
```
"Vous voulez le [template/checklist/framework] complet ? Envoyez-moi '[keyword]' en DM et je vous l'envoie."
```

## Comment Gating (Highest Engagement CTA)

Les posts comment-gatés génèrent 3-5x le volume de commentaires normal. L'algorithme voit des commentaires massifs → boost de distribution.

**Keyword Comment Gate :**
```
Commentez "INTENT" et je vous partage tout.
Commentez "STACK" et je vous envoie le breakdown complet.
Commentez "CLAUDE" et assurez-vous qu'on est connectés pour que je puisse vous DM.
```

**P.S. Question Closers (les plus performants) :**
```
P.S : Quel outil devrait-on ajouter à ce stack ?
P.S : Comment se portent vos campagnes en ce moment ?
P.S : Quel est l'outil dont vous ne pourriez plus vous passer ?
P.S : Sur quelle partie de ce système voulez-vous plus de détails ?
```

**Continuation Tease :**
```
Je manque de place... Je continue en commentaire 👇
Suivez pour ne pas rater le prochain breakdown.
```

## Profile as Conversion Funnel

Votre profil est une landing page. Chaque élément doit convertir les visiteurs en followers, connections ou leads.

### Headline Formulas (220 caractères max)

| Formule | Exemple |
|---------|---------|
| "J'aide [audience] [résultat]" | "J'aide les PME à automatiser leurs opérations avec l'IA" |
| "Rôle \| Niche + Preuve" | "CEO Flowmatter \| IA pour PME \| 20K€ économisés par client" |
| "Rôle \| Ce que vous faites \| Preuve sociale" | "Consultant IA \| Automatisation pour TPE/PME \| Ex-Anthropic" |

**À éviter :** "visionnaire", "guru", "ninja", "thought leader", ou titre de poste seul.

### About Section Structure

```
[HOOK — 2 premières lignes visibles avant "voir plus"]
[PROBLÈME QUE VOUS RÉSOLVEZ — 2-3 phrases]
[VOTRE APPROCHE — 3-5 phrases]
[PREUVES — 3-5 bullet points avec résultats spécifiques]
[AVEC QUI JE TRAVAILLE — 2-3 phrases]
[CTA — prochaine étape exacte : DM, email, réserver un appel]
```

### Featured Section (Ordre de priorité)

1. Lead magnet / ressource gratuite (développe la liste email)
2. Meilleur post LinkedIn (preuve sociale)
3. Étude de cas / témoignage (preuve de résultats)
4. Newsletter (croissance des abonnés)
5. Apparition média (autorité)

Max 3-5 éléments. Le plus important = le plus à gauche. Mise à jour mensuelle.

### Banner Design

- Dimensions : **1584 x 396 pixels** (ratio 4:1)
- Gardez le texte critique au centre (bords coupés sur mobile)
- Incluez : tagline, URL du site, offre clé, ou preuve sociale
